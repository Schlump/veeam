# Veeam Client

Veeam Rest API Client or wrapper to make it easier to interact with the Veeam API.

[The Veeam API documentation](https://helpcenter.veeam.com/backup/rest/overview.html)

## Installation

    pip install veeam

## Usage

    from veeam.client import VeeamClient
    
    client = VeeamClient()

## Running Tests

    pytest
